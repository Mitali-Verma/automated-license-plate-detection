# Dataset Information

The dataset used in this project consists of real-world license plate
images collected for experimental evaluation of ALPR systems.

Due to privacy, licensing, and ethical constraints, the dataset
not published here.

---

## Dataset Description

- Total images: 588
- Image type: Vehicle images containing visible license plates
- Variations:
  - Multiple lighting conditions
  - Different camera angles
  - Motion blur and occlusion
  - Diverse plate formats

---

## Annotations

- No character-level ground truth annotations are available
- Evaluation is performed at the plate level
- Metrics include:
  - Plate detection rate
  - Confidence scores
  - Format validation
  - Failure rate

---

## Note

Results reported in this repository are based on the private dataset
described above. The provided code and notebooks are shared for
transparency and methodological reference.
